**Author:** Yingshan Chen (Northwestern University, Data Science & Business Leadership)  
**Course:** STAT 390 Capstone  
**Timeline:** April 2026 – June 2026

## Project Overview
This project investigates whether an autonomous AI agent can identify inefficiencies in the NBA betting market. By leveraging a historical dataset of over 23,000 games, I use an **AutoResearch loop** (inspired by Andrej Karpathy) to iteratively propose, code, and evaluate statistical features that predict outcomes "Against the Spread" (ATS).

### The Research Question
> "Can an autonomous AI research loop identify specific historical game conditions (e.g., rest advantage, road-trip fatigue, or early-game scoring efficiency) where the closing point spread fails to accurately reflect the final scoring margin?"

---

## 🛠 Repository Structure
This repository follows a strict "Agent-Centric" workflow to ensure reproducibility and rigor.

- `data/raw/`: **[FROZEN]** Original `nba_2008-2025.csv` dataset.
- `data/processed/`: **[MODIFIABLE]** Datasets generated by the agent with engineered features.
- `agents/`: Core logic for the **Planner**, **Coder**, and **Critic** agents.
- `src/`: 
    - `baseline.py`: Initial Logistic Regression benchmark.
    - `evaluation.py`: **[FROZEN]** Standardized scoring logic to prevent agent "cheating."
- `experiments/`: JSON logs and performance history for every agent iteration.
